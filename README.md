# Portafolio

Portafolio de Juan Antonio

## 🚀 Tecnologías

- Astro
- Tailwind CSS
- Prism

## 📦 Instalación

```bash
npm install
```

## Comenzar

```bash
npm run start
```

## 👨‍💻 Desarrollo

```bash
npm run dev
```

## 📦 Compilación

```bash
npm run build
```
